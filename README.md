## [<img width="200" src="./qr.png">](https://labify-delta.vercel.app/)

### Generate labs from video lecture

## Installation

    ```bash
    $ git clone
    $ cd labify
    $ vercel env pull .env
    $ npm install

    $ pnpm dlx prisma generate
    ```

## Prisma Commands

    ```bash
    $ pnpm dlx prisma db push
    $ pnpm dlx prisma studio
    $ pnpm dlx prisma generate

    $ pnpm dlx prisma migrate dev --name init
    $ pnpm dlx prisma db pull
    ```

## Running the app

    ```bash
    # development
    $ npm run dev

    # production mode
    $ npm run build
    $ npm run start
    ```

## Test

    ```bash
    # unit tests
    $ npm run test

    # e2e tests
    $ npm run test:e2e

    # test coverage
    $ npm run test:cov
    ```
